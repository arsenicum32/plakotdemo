# Plakot

Крутой сборщик сайта plakot для того, чтобы нормально провалидировать все идеи.

---

## Запуск

```
npm run dev
```

## Сборка в папку dist

```
npm run build
```

## Деплой на demo.plakot.ru

```
npm run deploy
```

В конфиге `deploy.sh` нужно поменять никнейм на свой.
Если отказывается запускаться: `chmod +x deploy.sh`

---

## Структура src

- `styles/` — только глобальные стили
- `components/` — представление без логики, каждый компонент в отдельной папке со стилем
- `containers/` — подключение компонентов к redux store
- `redux/` — actions, reducers, store

### Соглашения по стилям

- CSS-модули через динамические классы: `import { needClass } from './style.styl'`
- По БЭМ: не прописывать margin у наружного блока, матч только по классам
- Все значения из переменных в `../styles/theme.styl`
